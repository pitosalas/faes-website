# Feature description for feature F15
## F15 — Language switcher for bilingual pages
**Priority**: High
**Done:** yes
**Tests Written:** yes
**Test Passing:** yes
**Description**: Separate bilingual content into individual English and Papiamentu markdown files. Add a language switcher button in the page template that allows users to toggle between English and Papiamentu versions of home, mission, and about pages. The English page will be the default, with a button to switch to the Papiamentu version and vice versa.
